    ConditionalAE

    Yoojin Choi, Mostafa El-Khamy, Jungwon Lee, “Variable Rate Deep Image Compression With a Conditional Autoencoder,” ICCV-2019

    Introduce two rate control parameters

    1. Lagrangian multiplier $\lambda$ (coarse rate adaptation)
      • $\lambda$ is chosen from a predefined finite set of values.
      • It is injected to the network through conditional convolution block (for both encoder, decoder and prior model). During training $\lambda$ is randomly sampled
    2. Quantization bin size $\Delta$ (fine rate adaptation)
      • Only the quantizer and prior probability evaluation take $\Delta$ as input

    Remarks

    • Main idea is to solve multiple objectives using one conditional network, instead of solving them individually using separate non-conditional networks.
    • Since both $\lambda$ and $\Delta$ are parameters to the decoding process, they need to be included as part of the bitstream and decoded first.
    • Universal quantization with dithering is applied instead of adding uniform noise to approximate quantization noise during training, and all latent are dithered with a single uniform random variable. Authors reported better performance with this dithering based universal quantization, but no intuition/reasoning is provided.
    • Single model performs on par with JointARHP and ContextAdapt in the End-to-End image compression post

    $\lambda$-dependent channel-wise scaling and offset

    QualityScalingFactor

    Tong Chen, Zhan Ma, “Variable Bitrate Image Compression with Quality Scaling Factors”, ICASSP-2020

    Motivating experiment/observation

    • First train a simple VAE network similiar to at a high bitrate, and retrained this network to get multiple models at different but lower bitrates.
    • An interesting observation is that similar but scaled patterns are retained for feature maps at a specific channel across a variety of bit rates, leading to a reasonable conclusion that we could apply the scaling after the identical convolutions without resorting to the complete retraining (see right figure below).

    Idea/Remarks

    • Finetune per-channel quantization steps for each R-D point. Encoder, decoder, prior model are fixed.
    • Per-channel quantization steps need to be transmitted to the decoder as well (not mentioned in the paper).
    • This scheme is similar to setting $\Delta$ in ConditionalAE.
